CSS BOX MODEL..!

Every element that can be displayed on a weeb page is comprised of one or more rectangular boxes. Css box model typically describes how these rectangular boxes are laid out of a web page. These boxes can have different properties and can interact with each other in diferent ways, but every box has a content area and optional surrounding padding, border and margin areas.
